# Slide Visual Reference Prompt Pattern

Use visual generation only when it adds real value. The four-slide sample must be approved before full-deck production.

## Prompt Pattern

Create a 16:9 editorial-academic slide visual reference for a medical, nursing, public-health, or behavioral-science presentation.

Visual direction:

- warm off-white background;
- muted teal primary color;
- restrained coral accent;
- lightweight navigation tabs;
- clean editorial composition;
- intentional negative space;
- minimal borders and no heavy shadows;
- no dashboard-like card grid;
- no more than three cards if cards are genuinely necessary;
- use one of these structures: editorial two-column, large evidence + interpretation, full-width logic chain, comparison matrix, or single-conclusion page.

Content intent:

- Slide purpose: [PURPOSE]
- Main takeaway: [TAKEAWAY]
- Main evidence/visual: [EVIDENCE]
- Secondary explanation: [EXPLANATION]
- Reading path: [READING PATH]

Constraints:

- Do not generate exact statistics, P values, confidence intervals, formal tables, or dense article text.
- Do not fabricate charts.
- Use abstract placeholders only.
- Keep the navigation visually subordinate to the title and evidence.
- Avoid large empty color blocks used only to fill space.
- The output is a layout reference, not the final slide.
