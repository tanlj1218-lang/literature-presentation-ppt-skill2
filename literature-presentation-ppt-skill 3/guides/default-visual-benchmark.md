# Default Visual Benchmark

## Purpose

This benchmark records the user's preferred visual direction from the first literature-presentation version. It is the default style when the user does not provide a newer approved template.

Reference images:

- `assets/default-visual-benchmark/title-page.png`
- `assets/default-visual-benchmark/introduction-page.png`
- `assets/default-visual-benchmark/method-page.png`
- `assets/default-visual-benchmark/result-page.png`

## Core Characteristics

- warm off-white background;
- compact, lightweight teal navigation tabs;
- dark teal titles;
- restrained coral accent;
- editorial rather than dashboard-like composition;
- limited use of borders and rounded cards;
- clean tables with teal headers;
- strong left alignment;
- generous but intentional negative space;
- figures shown at a readable scale;
- citation line kept quiet but legible.

## What Must Be Preserved

- the visual calmness;
- title and evidence hierarchy;
- lightweight navigation;
- restrained palette;
- consistent margins and alignment;
- clean editorial rhythm.

## What May Be Improved

- add more external evidence and theory where academically necessary;
- improve figure annotation and source notes;
- enlarge result figures;
- vary page archetypes while preserving the same visual language;
- reduce repetitive divider pages;
- improve citation legibility.

## What Must Not Happen

- replacing the style with a dense card dashboard;
- adding large colored containers simply to fill space;
- turning the top navigation into a heavy dark banner;
- using tiny screenshots;
- allowing content expansion to trigger an unapproved redesign.
