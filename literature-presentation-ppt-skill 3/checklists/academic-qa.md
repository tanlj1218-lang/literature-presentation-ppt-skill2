# Academic QA Checklist

## Bibliographic Information

- [ ] Title, journal, DOI, dates, authors, affiliations, funding, and registration are correct.
- [ ] Impact factor, JCR quartile, and CAS partition include year/version and source.

## Scientific Narrative

- [ ] The introduction follows one coherent line and is detailed enough for a 45-minute presentation.
- [ ] The introduction/theory page count is driven by scientific need rather than a quota.
- [ ] Every substantive introduction slide has evidence.
- [ ] Burden, concept, theory, current evidence, methodological gap, and scientific gap are distinguished.
- [ ] The gap is specific.
- [ ] Aims/hypotheses are accurately stated.

## Theory

- [ ] Original theory source was checked.
- [ ] Each theory explains a specific part of the paper.
- [ ] Constructs and pathways are accurate.
- [ ] Theory is mapped to variables, measures, population, design, and method.
- [ ] Applicability and limits are stated.
- [ ] Complementary theories are included only when they materially improve interpretation.

## Evidence Search

- [ ] Search terms, dates, and sources are documented.
- [ ] Evidence includes original theory/method papers and recent reviews.
- [ ] References were not copied blindly from the target paper.
- [ ] Each cited source supports the exact claim.
- [ ] Every substantive slide has a visible source marker.

## Methods

- [ ] Design, recruitment, sample, criteria, timing, and quality control are accurate.
- [ ] Missing-data handling and covariates are reported.
- [ ] Instruments and scoring are verified.
- [ ] Statistical methods, assumptions, parameters, diagnostics, and limitations are accurate.
- [ ] External methodological recommendations are clearly distinguished from article facts.

## Results

- [ ] Participant numbers and descriptive values match the paper.
- [ ] Effect sizes, confidence intervals, P values, and figure/table sources are correct.
- [ ] Statistical significance is not confused with practical importance.
- [ ] Association is not presented as causation.
- [ ] Simulation findings are not presented as validated interventions.
- [ ] Stability, sensitivity, and robustness results are included when important.

## Discussion

- [ ] Main findings answer the research questions.
- [ ] Agreement/disagreement names and cites the comparison study.
- [ ] Mechanisms are supported or labeled as inference.
- [ ] Author limitations and independent appraisal are distinguished.
- [ ] Generalizability and translation are appropriately limited.

## User-Research Implications

- [ ] Suggestions are specific to the user's research.
- [ ] Transferable and non-transferable elements are separated.
- [ ] Parameters are not copied without checking suitability.
- [ ] Proposed extensions are methodologically plausible.

## Traceability

- [ ] Every key number and figure/table has a source location.
- [ ] Evidence tracker is complete.
- [ ] Unresolved issues are disclosed.
