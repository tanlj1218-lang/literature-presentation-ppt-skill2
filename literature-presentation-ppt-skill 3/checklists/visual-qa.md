# Visual QA Checklist

## Global Consistency

- [ ] 16:9 ratio is correct.
- [ ] Microsoft YaHei is used for Chinese text.
- [ ] Top navigation is consistent.
- [ ] Active navigation item is correct on every slide.
- [ ] Main, supporting, accent, and neutral colors are consistent.
- [ ] Margins and grid are consistent.
- [ ] Footer and page numbering are consistent.

## Typography

- [ ] Titles are readable from a distance.
- [ ] Body text is not too small.
- [ ] Reference text remains readable.
- [ ] Line spacing is consistent.
- [ ] No text overflow or clipping.
- [ ] No garbled Chinese characters.
- [ ] Bold and accent color are used sparingly.

## Figures and Tables

- [ ] Figures are high resolution.
- [ ] Figures are not stretched or distorted.
- [ ] Tables are legible.
- [ ] Key result is visually emphasized without hiding context.
- [ ] Recolored charts preserve original meaning.
- [ ] No misleading axis or visual scale.
- [ ] Source label is present.
- [ ] Every substantive slide has a reference footer.
- [ ] Reference footer is readable after slide rendering.

## Layout

- [ ] No overlapping objects.
- [ ] No objects clipped at slide edges.
- [ ] Alignment is precise.
- [ ] Spacing is balanced.
- [ ] Ordinary content slides use approximately 70–88% of the main content area.
- [ ] No single unintentional blank region occupies more than approximately 18% of the slide.
- [ ] Content is not concentrated in one quadrant.
- [ ] Each ordinary slide contains at least two meaningful content units.
- [ ] Left and right columns have comparable visual weight.
- [ ] Top, middle, and bottom zones are intentionally used.
- [ ] Decorative assets do not compete with evidence.
- [ ] Result slides follow one-main-conclusion principle.
- [ ] Left-evidence/right-interpretation layout is used where appropriate.

## Generated Assets

- [ ] Generated assets have transparent backgrounds when required.
- [ ] No white halo or watermark.
- [ ] No pseudo-text or garbled labels.
- [ ] Generated visual reference is not used as a flattened full-slide background.
