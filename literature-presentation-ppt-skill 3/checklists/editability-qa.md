# Editability QA Checklist

- [ ] All formal text is in native text boxes.
- [ ] Chinese text remains editable and correctly encoded.
- [ ] Simple blocks, lines, arrows, and shapes are native objects.
- [ ] Tables are native tables or editable shape groups.
- [ ] Charts are native charts when exact data are available.
- [ ] Unavoidable source figures are separate image objects.
- [ ] No entire slide is a single flattened background image.
- [ ] Objects are logically grouped.
- [ ] Layer order is correct.
- [ ] Transparent assets behave correctly.
- [ ] Objects can be selected and moved without hidden overlays.
- [ ] Chart data can be edited where applicable.
- [ ] No embedded font files are included.
- [ ] PPTX opens without repair warnings.
