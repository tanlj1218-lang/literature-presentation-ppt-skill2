# Per-Slide Image2 / Image_Gen Visual Reference Prompt

Use this prompt for **every individual slide** after the full-deck blueprint and slide specification are approved. Do not use one generic prompt for the whole deck.

## Prompt template

Create one complete 16:9 academic slide visual reference for a medical, nursing, public-health, psychology, or behavioral-science literature presentation.

### Page-specific scientific logic

- Slide number: [NUMBER]
- Section: [SECTION]
- Scientific question answered: [QUESTION]
- Conclusion-style title intent: [TITLE INTENT]
- Main takeaway: [TAKEAWAY]
- Previous page established: [PREVIOUS]
- This page adds: [CURRENT]
- Next page transition: [NEXT]

### Content and evidence hierarchy

- Dominant evidence object: [PAPER FIGURE / FIGURE PANEL / NATIVE TABLE / CONCEPTUAL MODEL / LOGIC CHAIN / KEY NUMBER]
- Exact evidence aspect ratio or reserved area: [DETAIL]
- Interpretation sequence: [HOW TO READ → KEY RESULT → MEANING → LIMITATION]
- Required secondary elements: [SHORT LABELS / DEFINITIONS / CALLOUT / SOURCE NOTE]
- Reference/footer area: [LOCATION]

### Visual direction

- rigorous editorial-academic style;
- warm off-white background;
- muted teal primary color;
- restrained coral accent;
- dark teal/charcoal text hierarchy;
- lightweight top navigation;
- clean alignment and meaningful negative space;
- evidence stronger than decoration;
- one of these page structures: editorial two-column, large evidence + interpretation, full-width logical chain, comparison matrix, or single-conclusion page;
- coherent with the approved title, introduction, methods, and results samples.

### Mandatory restrictions

- This image must reflect this slide's actual scientific argument, not a generic medical theme.
- Do not invent statistics, P values, confidence intervals, sample sizes, table values, or scientific charts.
- Do not rely on generated text for the final slide; use short layout labels only.
- Do not create fake article screenshots.
- Do not add unrelated DNA, microscope, hospital, patient, brain, or healthcare icons.
- Do not use dashboard-style card grids or excessive decoration.
- For result slides: one figure or one coherent figure panel + one interpretation block only.
- Keep the image suitable for element-level reconstruction into editable PowerPoint objects.

The output is a full-slide visual blueprint. It will be recreated element by element in PowerPoint and must not be used as a flattened background.
