# Storyboard and Page-Logic QA

- [ ] A full-deck blueprint exists before full visual production.
- [ ] Every slide, including divider, references, and appendix pages, is listed.
- [ ] Every slide has a conclusion-style title or explicit page function.
- [ ] Every slide answers one scientific question.
- [ ] Every slide has one dominant takeaway.
- [ ] Previous → current → next logic is documented for every page.
- [ ] Exact content blocks and evidence sources are specified.
- [ ] Page order forms one coherent scientific argument.
- [ ] Introduction moves from importance to evidence gap to study response.
- [ ] Methods follow the analysis workflow rather than arbitrary categories.
- [ ] Results follow the research questions and analysis sequence.
- [ ] Discussion maps findings back to questions, evidence, mechanisms, and limitations.
- [ ] The full page list was shown to the user and approval was recorded.
