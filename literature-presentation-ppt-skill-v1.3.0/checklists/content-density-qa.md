# Visual Balance, Density, and Citation QA Checklist

## Meaningful Density

- [ ] Content depth comes from evidence, explanation, comparison, or mechanism—not from extra boxes.
- [ ] A sparse slide is retained only when the negative space is intentional.
- [ ] A weak slide was merged when it did not carry a complete point.
- [ ] An overcrowded slide was split rather than shrunk.
- [ ] A page is not judged by a fixed occupancy percentage.

## Anti-Cardification

- [ ] Cards are used only when they represent real categories or comparisons.
- [ ] No more than three cards appear on a normal slide.
- [ ] Consecutive card grids are avoided.
- [ ] Large pale containers are not used merely to occupy space.
- [ ] Position, scale, spacing, and typography create hierarchy before borders are added.

## Projection Readability

- [ ] Body text is readable in presentation mode.
- [ ] References are readable after rendering.
- [ ] Figures and tables can be understood without zooming.
- [ ] Dense source figures were enlarged, cropped, redrawn, split, or moved to the appendix.
- [ ] No tiny thumbnail is used as decorative evidence.

## Citation Coverage

- [ ] Every substantive slide has a visible source marker.
- [ ] Introduction/theory/method/discussion slides contain concise readable citations.
- [ ] Result slides cite the target paper and figure/table number.
- [ ] Claims of agreement or disagreement name the comparison study.
- [ ] Mechanism claims are supported by external evidence.
- [ ] Full references are included at the end or in the delivery package.

## Final Montage Review

- [ ] The deck was reviewed as a montage.
- [ ] Repetition, navigation weight, card overuse, color drift, and unreadable visuals were corrected.
- [ ] Final deck remains consistent with the approved style sample.
